
# DEVFLIX

A Devflix é uma plataforma de filmes e séries, inspirada na Netflix, focada em inovação e desenvolvimento. Trazendo breves descrições sobre o filme escolhido.
## Funcionalidades

- Temas dark e light
- Busca de filme desejado
